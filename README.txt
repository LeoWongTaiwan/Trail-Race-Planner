Trail Planner 終極 UI 原型

核心體驗
- 首頁三層架構：
  1. 深藍公告欄：今天、快截止、新開放
  2. 米黃色推薦：依距離、UTMB、交通便利度
  3. 白色 Explore：所有搜尋與篩選入口
- 固定底部導航，避免上一頁混亂
- Explore 可收合篩選：距離、月份、地區、賽事條件
- 距離區間：10–20K、20–50K、50–70K、70–100K、100K+
- 比賽日期與報名截止日期雙向排序
- 離廣州近到遠
- 一場賽事含多個距離組別
- 完整賽事詳情頁：組別、爬升、交通、天氣、重要日期
- 我的賽季：想跑、已報名、已完賽
- 偏好與提醒設定
- 可安裝到手機主畫面

目前為高保真模擬資料版，用於最終確認 UI。
將 zip 上傳 Netlify 即可測試。
